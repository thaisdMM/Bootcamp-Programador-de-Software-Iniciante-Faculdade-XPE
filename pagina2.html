Seja bem vindo a página 2!

Criei essa página 2 para ser link com o meu index.html que explica que essa página pode ser adicionada com um elemento "a": '< a href="...">'. Assim, para o link dessa página tem que escrever o seguinte:

a href="./pagina2.html">link para outro documento a

A página 2 também foi adicionada ao meu index.html por meio da abertura de um link em nova aba com o atributo target="_blank". Assim, para o link dessa página tem que escrever o seguinte:

a href="pagina2.html" target="_blank">Página 2 a

Não se esqueça de utilizar: <> entre o "< a href='...' >" e para fechar o < "a" />.
